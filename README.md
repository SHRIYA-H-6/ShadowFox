# Boston House Price Prediction

## Description
This project predicts house prices using Linear Regression.

## Features Used
- Crime rate
- Number of rooms
- Tax rate
- etc.

## Technologies
- Python
- Pandas
- Scikit-learn

## How to Run
1. Install dependencies:
   pip install pandas scikit-learn

2. Run:
   python house_price_prediction.py

## Output
- Predicted house prices
- MAE and R2 score
